# Table Editor - Frontend Assignment

A React + TanStack Table + Tailwind app that allows users to:
- View table records
- Edit entries
- Update via API
- Country dropdown via API
- Modal UI & smooth UX

## Live Demo
https://table-editor--sujaldonga27.replit.app

## Tech Stack
React
Vite
TailwindCSS
@tanstack/react-table
Axios

## Run Locally
npm install
npm run dev
